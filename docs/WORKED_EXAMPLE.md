# Worked Example Design

The LHC black-hole case is useful because a shallow system can produce a good
debate map while missing the physical argument.

## Shallow Layer

The shallow layer answers:

- who argued for safety;
- who challenged the conclusion;
- which papers cite or respond to each other;
- what the verbal cruxes are.

This is not wrong.  It is incomplete.

## Mechanism Layer

The mechanism layer asks whether the source set contains witnesses for these
branches:

```text
production threshold
  -> evaporation branch
  -> stable branch
       -> capture/stopping
       -> accretion/growth
       -> astrophysical bound
       -> exclusion conclusion
```

The important output is not a fluent answer.  It is a set of source-local
equations, contexts and role labels that can be inspected.

## Failure Criterion For A Provenance-Only System

A provenance-only system fails when it can identify the controversy but cannot
state which physical branch is broken by a challenge paper.

For example, a challenge to Hawking evaporation does not automatically imply
danger.  It moves the argument to the stable-object branch, where capture,
accretion and astrophysical survival bounds must be evaluated.
